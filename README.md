
# Unit 1 — Egypt–Africa Cooperation (v4.1)

Interactive mini platform for 2ND SEC TERM 1 — Unit 1.

## What’s inside
- Single-page **index.html** (offline-ready, mobile friendly)
- Grammar expanded (Present Continuous — Active & Passive)
- Unit Exam: **30 contextual sentences** (15 vocab in context + 15 grammar)
- Matching game speaks the word on correct match
- Review mode (shows your answer, correct answer, and hint)

## How to publish to GitHub Pages
1. Create a new repository (e.g., `unit1-v4-1`).
2. Upload **index.html** to the root of the repo.
3. Go to **Settings → Pages** → Source: **main** branch, Folder: **/**.
4. Click **Save**. Your site will be available at:
   `https://<your-username>.github.io/unit1-v4-1/`

> No external assets are required; it’s pure HTML/CSS/JS.
